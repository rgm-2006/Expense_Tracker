Expense Tracker System

A simple Java-based Expense Tracker application that helps users manage their daily expenses using file handling.

 Features

- Add new expenses
- View all expenses
- Calculate total expenses
- Store expense data in a text file
- Simple console-based interface

 Technologies Used

- Java
- File Handling
- OOP Concepts
- BufferedReader & FileWriter
 Project Structure

```text
ExpenseTracker.java
expenses.txt
README.md
```

 How It Works

The application stores expense details in a text file named:

```text
expenses.txt
```

Each expense contains:
- Date
- Category
- Amount
- Description
